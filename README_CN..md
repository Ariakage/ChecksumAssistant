### 简介
使用Golang开发，为文件生成校验码
### 使用
在终端中输入 `checksumassistant [filename]` 接下来程序将会输出
```
Welcome to use ChecksumAssistant!
Processing file: [filename]
MD5:      xxx
SHA1:     xxx
SHA256:   xxx
CRC32:    xxx
```
该程序支持下述的校验码类型:
- [x] MD5
- [x] SHA1
- [x] SHA256
- [x] CRC32
